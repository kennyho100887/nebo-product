# Nebo Health — UM Nurse Prototype

A product redesign prototype for the Nebo Health Utilization Management (UM) nurse workflow, by **Konpo Studio**.

This is a **self-contained HTML file** — no build step, no external CDN. Open [`nebo_um_nurse_prototype.html`](nebo_um_nurse_prototype.html) directly in any modern browser.

## Screens

- **My Queue** — the nurse's personal worklist of assigned cases.
- **All Cases** — the full case list across the team.
- **New Case** — a modal for intake of a new case.
- **Case Details / Assessment** — the per-case detail and clinical assessment view.

## Notes

- Tabler icons are used as a subset and inlined as base64, so the prototype renders fully offline with no network requests.
